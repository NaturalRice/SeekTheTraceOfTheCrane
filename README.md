游戏WebGL打包程序链接：https://naturalrice.github.io/SeekTheTraceOfTheCrane-WebGL/
点开网址就直接进入游戏了，然后按Del退出游戏。由于方便测试玩家移动比较快，在NPC面前按空格键进行对话，然后背包里的东西可以鼠标选中移动到物品栏，
鼠标左键聊天记录框内按住可滑动聊天栏。
（有一个锄头在物品栏按数字键选中后，人物移动到下面淡蓝色的区域按空格键可以改变地图贴图，后续这个功能可能用在修复文物的显示功能上）
